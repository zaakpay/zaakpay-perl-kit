Hello Dear User,
This is a ReadMe for the Zaakpay Perl kit.
TRANSACT API : This is the primary API used to perform a transaction.
It makes use of the following files.
test_merchant_input.html : This is the HTML file which takes input from user & posts data.
posttozaakpay.pl : This is the file which accepts input, sanitizes it & posts it to the TRANSACT API.
response.pl : The TRANSACT API completes the transaction & redirects user to this file.

To get going with Perl Kit :
  1. Enter the Merchant Identifier in the test_merchant_input.html file
  2. Enter the secret key in both the response.pl and posttozaakpay.pl
  3. Now,you are good to go with transactions

Note :
  1. To perform test transaction , try mode =0
